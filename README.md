# prtg-php

Using this library you can get details on your sensors , create charts and more

## Install
### Using Composer
Add this package to your `composer.json`:
```json
"require": {
    "satrobit/prtg-php": "dev-master"
}
```
or by command line :
```bash
composer require satrobit/prtg-php
```
